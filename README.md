Mini-projeto voltado para a matéria de Testes, aqui nele aplicaremos os princípios que aprenderemos. 
Primeira Parte:
Cada integrante do grupo deverá construir no mínimo 20 testes unitários. Lembrem-se de realizar suite testes/ integração.

Requisitos para as equipes que não utilizarem o TCC:

Desenvolver um sistema
1 - Tela de Login
2 - Usuário Admin
3 - Cadastro de usuário (Nome completo, ID, E-mail, Telefone)
4 - Remoção de usuário
5 - Cadastro de produtos (3 tipos)
6 - Remoção de produtos
7 - Compra e Venda
8 - Atualização de estoque
9 - Teste Unitário/Integração

Lembrem-se de construir um sistema capaz de evoluir para uma interface gráfica.
Segunda Parte Requer:
• Criar no mínimo 30 casos de teste (Por aluno)
• Incluir testes:
o Funcionais (ex: login, cadastro, exclusão)
o De validação (ex: campos obrigatórios, formatos inválidos)
• Utilizar linguagem clara e objetiva
• Garantir que os testes possam ser reproduzidos por outra pessoa

Entrega
• Entregar a planilha preenchida
• Relatório curto contendo:
o Principais erros encontrados
o Dificuldades enfrentadas
o Sugestões de melhoria no sistema
